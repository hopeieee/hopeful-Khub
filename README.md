# Hopeful KHub (K-pop Universe)
Founded by Hopedeep Kaur. The ultimate interactive fan app for K-pop lovers.

## 🚀 How to Run Locally
1. **Clone the repo**: `git clone <your-repo-url>`
2. **Install dependencies**: `npm install`
3. **Set up environment variables**: Create a `.env` file with your `GEMINI_API_KEY`.
4. **Run the app**: `npm run dev`

## ✨ Features
- **Virtual Stage**: Real-time concert experience with Socket.io.
- **AI Chatbot**: Powered by Google Gemini.
- **Photocard Creator**: AI-enhanced digital collectibles.
- **Bias Finder**: Personalized idol matching.
